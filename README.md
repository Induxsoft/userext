# Userext (Usuarios externos).
El paquete Userext (Usuarios externos) de V12 es el módulo que permite administrar la información y credenciales de usuarios externos a la organización, garantizando un acceso controlado y seguro a los sistemas del ERP. Su propósito es facilitar la colaboración con terceros —como clientes, proveedores o socios— sin comprometer la integridad de los datos internos.
